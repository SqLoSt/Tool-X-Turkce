- **Eğer tek seferde kurmak ister isen** :
basitçe kodu terminale yapıştır :
``` 
apt update && apt install git && git clone https://github.com/ekadanuarta/Tool-X.git && cd Tool-X && chmod +x install && sh install 
```

```

cd Tool-X
chmod +x install
```
